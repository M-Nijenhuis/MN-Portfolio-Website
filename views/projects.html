Todo app

Dit een kleine todo webapp. Dit project was vooral veel gefocust op de database en de backend functionaliteit. Maar ik vond het wel leuk om op mijn portfolio te zetten. Het is gewoon een simpele todo app. Ik heb voor de strakke stijl gekozen. Je kan todo's toevoegen, verwijderen, veranderen en afvinken. Ik heb ook de tijd toegevoegd als een onderdeel van het todo item. Als de echte tijd voorbij de tijd van het todo item is en je vinkt het item af, dan word het verwijdert. Maar als de echte tijd nog niet de tijd van het todo item is geweest, dan blijft het staan.

Mijn portfolio website
